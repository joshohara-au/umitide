# Apps Folder

This folder will contain all BiteBack applications and modules.

## Structure
- `frontend/` - Mobile app (React Native) for BiteBack
- `backend/` - Node.js + Python services
- `ai_agents/` - Autonomous agents for RAG, predictions, and task automation
- `mini_game/` - 3D interactive mini-game files
- `assets/` - Mascot 2D/3D assets, UI mockups

Each folder will include its own README and setup instructions.

This is the initial commit to create the `apps` folder structure for BiteBack.
