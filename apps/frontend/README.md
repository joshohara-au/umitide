# Frontend App

This folder contains the BiteBack mobile application.

## Structure
- `src/` - Source code for the React Native app.
- `assets/` - Images, icons, and mascot sprites.
- `components/` - UI components.
- `screens/` - App screens and navigation.
- `services/` - API services for backend and AI integration.

This is the initial placeholder to create the `frontend` folder inside `apps`.
